# envoc.chatter

[![Get help on Codementor](https://cdn.codementor.io/badges/get_help_github.svg)](https://www.codementor.io/justinobney)

## a slack like app used to learn firebase

Features

* users
  * profile
  * online presence
* channels
  * `@` mentions
  * 'username' is typing
* commands
  * close
  * kick @username
  * ask `@username` question
* preferences
  * highlight words
* chatterbot!
* decisions
* hotkeys
  * `ctrl+k`
